UI is a component library which is light weight, easy to use, out of the box modern styling. Stop worrying and start applying!

Installation Guide

Include the <link/> tag given below in your <head/> tag above all the stylesheets. 
                            <link rel="stylesheet" href="https://ui-components-library.netlify.app/css/component.css"/>
Some components may require use of JavaScript, so include below code in <body/> tag at the very end. 
                            <script src="https://ui-components-library.netlify.app/index.js"></script>
